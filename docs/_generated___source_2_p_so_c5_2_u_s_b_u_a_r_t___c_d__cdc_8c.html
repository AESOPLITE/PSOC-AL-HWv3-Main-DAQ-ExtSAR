Returns
requestHandled

\globalvars USBUART_CD_linesCoding: Contains the current line coding structure. It is set by the Host using SET_LINE_CODING request and returned to the user code by the USBFS_GetDTERate(), USBFS_GetCharFormat(), USBFS_GetParityType(), USBFS_GetDataBits() APIs. USBUART_CD_linesControlBitmap: Contains the current control signal bitmap. It is set by the Host using SET_CONTROL_LINE request and returned to the user code by the USBFS_GetLineControl() API. USBUART_CD_linesChanged: This variable is used as a flag for the USBFS_IsLineChanged() API, to be aware that Host has been sent request for changing Line Coding or Control Bitmap.

\reentrant No.
