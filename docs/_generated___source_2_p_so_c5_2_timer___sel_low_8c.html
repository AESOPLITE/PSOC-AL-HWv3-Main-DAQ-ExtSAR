Description: The Timer component consists of a 8, 16, 24 or 32-bit timer with a selectable period between 2 and 2^Width - 1. The timer may free run or be used as a capture timer as well. The capture can be initiated by a positive or negative edge signal as well as via software. A trigger input can be programmed to enable the timer on rising edge falling edge, either edge or continous run. Interrupts may be generated due to a terminal count condition or a capture event.
