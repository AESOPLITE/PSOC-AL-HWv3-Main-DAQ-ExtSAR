void isr_W_SetPriority ( uint8  priority)

Function Name: isr_W_SetPriority

Summary: Sets the Priority of the Interrupt.

Note calling isr_W_Start or isr_W_StartEx will override any effect this API would have had. This API should only be called after isr_W_Start or isr_W_StartEx has been called. To set the initial priority for the component, use the Design-Wide Resources Interrupt Editor.

Note This API has no effect on Non-maskable interrupt NMI).

Parameters: priority: Priority of the interrupt, 0 being the highest priority PSoC 3 and PSoC 5LP: Priority is from 0 to 7. PSoC 4: Priority is from 0 to 3.
