Vendor Specific Declarations Function Name: USBUART_CD_HandleVendorRqst This routine provide users with a method to implement vendor specific requests.

To implement vendor specific requests, add your code in this function to decode and disposition the request. If the request is handled, your code must set the variable "requestHandled" to TRUE, indicating that the request has been handled.

requestHandled.
