void isr_Cm_SetPriority ( uint8  priority)

Function Name: isr_Cm_SetPriority

Summary: Sets the Priority of the Interrupt.

Note calling isr_Cm_Start or isr_Cm_StartEx will override any effect this API would have had. This API should only be called after isr_Cm_Start or isr_Cm_StartEx has been called. To set the initial priority for the component, use the Design-Wide Resources Interrupt Editor.

Note This API has no effect on Non-maskable interrupt NMI).

Parameters: priority: Priority of the interrupt, 0 being the highest priority PSoC 3 and PSoC 5LP: Priority is from 0 to 7. PSoC 4: Priority is from 0 to 3.
