CY_ISR ( SPIM_BP_RX_ISR  )

Function Name: SPIM_BP_RX_ISR

Summary: Interrupt Service Routine for RX portion of the SPI Master.

Parameters: None.

Return: None.

Global variables: SPIM_BP_rxBufferWrite - used for the account of the bytes which have been written down in the RX software buffer modified when FIFO contains new data. SPIM_BP_rxBufferRead - used for the account of the bytes which have been read from the RX software buffer, modified when overflow occurred. SPIM_BP_rxBuffer[SPIM_BP_RX_BUFFER_SIZE] - used to store received data, modified when FIFO contains new data. All described above Global variables are used when Software Buffer is used.
