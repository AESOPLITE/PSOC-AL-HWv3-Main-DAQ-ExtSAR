CY_ISR ( RTC_Main_ISR  )

Function Name: RTC_Main_ISR

Summary: This ISR is executed on 1PPS (one pulse per second) event. Global interrupt must be enabled to invoke this ISR. This interrupt trigs every second.

Parameters: None.

Return: None.

Global variables: RTC_Main_currentTimeDate, RTC_Main_dstTimeDateStart, RTC_Main_dstTimeDateStop, RTC_Main_dstTimeDateStart, RTC_Main_alarmCfgTimeDate, RTC_Main_statusDateTime, RTC_Main_dstStartStatus, RTC_Main_dstStartStatus, RTC_Main_dstStopStatus, RTC_Main_alarmCurStatus: global variables are used for the time/date, DST and alarm update procedure.

RTC_Main_dstTimeDateStart and RTC_Main_currentTimeDate: are modified with the updated values.

RTC_Main_statusDateTime, RTC_Main_dstStartStatus, RTC_Main_dstStartStatus, RTC_Main_dstStopStatus, RTC_Main_alarmCurStatus: global variables could be modified while current time/date is updated.

Side Effects: Clears all interrupt status bits (react_int, limact_int, onepps_int, ctw_int, and ftw_int) in Power Manager Interrupt Status Register. If an interrupt gets generated at the same time as a clear, the bit will remain set (which causes another interrupt).
